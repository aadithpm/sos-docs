Examples¶

Jupyter notebook to demonstrate features of SoS

  • Types of cells (notebook)
    Demonstrate four types of cells that can be included in a Jupyter notebook
  • DEG Annotation (notebook)
    An example to use Python to read an excel file, transfer the data to R, use R to translate ensembl gene IDs to HGNC gene names, then transfer data back to Python to write to a separate excel file
  • DEG Annotation with external task (notebook)
    The same example but submit the R part of the workflow as an external task
  • Generation of reports with hidden details (notebook and generated report)
    This example demonstrates how to generate a report in HTML format with hidden details
  • Simple External Tasks (notebook)
    An example to generate and execute a number of external tasks in Jupyter notebook. You can try to use option trunk_size to group the tasks. This example also demonstrates the use of option shared to collect results from tasks as variables
  • Remote Target (notebook)
    An example on the analysis of targets that reside on remote servers using target remote() and step option remote
  • Use SAS with SoS (notebook)
    Simple examples on using SAS with SoS
